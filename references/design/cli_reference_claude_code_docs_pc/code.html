CLI Commands Reference

Comprehensive guide to all available commands for the Claude Code desktop CLI. Use these tools to manage sessions, configuration, and developer workflows.

filter_list
key
claude auth Required

Manage authentication sessions. Log in to your Anthropic account to sync settings and enable premium features across devices.

forum
claude chat

Start a new interactive chat session in your terminal. Supports markdown rendering and multi-line input for complex queries.

settings
claude config

View or modify local configuration variables including model selection, output verbosity, and proxy settings.

system_update
claude update

Checks for new versions of the CLI and prompts for installation. Keeps your environment up-to-date with the latest AI improvements.

list_alt
claude logs

Access session history and debug logs. Useful for troubleshooting connectivity issues or reviewing previous AI responses.

lightbulb

Pro Tip

You can pipe the output of any command directly to a file or another utility. For example, use claude chat "explain this code" < file.py > explanation.txt to automate documentation tasks.
