Installing Claude Code

Get up and running with Claude Code on your local machine. This guide covers requirements, installation steps, and initial authentication.

1

Install Node.js

Claude Code requires Node.js version 18 or higher. We recommend using a version manager to handle multiple environments easily.

info

Recommended

Use nvm (Node Version Manager) to install Node.js. This avoids permission issues when installing global packages.

2

Run the command

Install the Claude Code CLI globally using your preferred package manager.

npm install -g @anthropic-ai/claude-code
bash — claude

$ claude dev

Initializing Claude Code v1.0.0...
? What would you like to build today?
_

3

Authenticate

Link your local environment to your Anthropic account to enable full functionality.

claude auth login

This will open a browser window for secure authentication. Once completed, your credentials will be stored securely in your system's keychain.
